## Atividade: Aula 11

# Redes de Computadores: Topologias, Dispositivos e Meios

Pesquisa com o objetivo de Entender a organização física e lógica das redes, identificar os principais dispositivos e reconhecer os diferentes meios de transmissão.

Esta pasta contém:

  - Diagramas de topologias com a representação das principais topologias de rede;
  - Quadro comparativo de dispositivos, comparando **Hub, Switch e Roteador**;
  - Classificação dos meios de transmissão em: **Guiados (com fio):** e **Não guiados (sem fio):** ,utilizando um esquema visual mostrando como os meios conectam os dispositivos.
