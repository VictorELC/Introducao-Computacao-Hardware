# Conceitos Fundamentais

### Internet x Web

A Internet é a infraestrutura física e lógica. Envolve os cabos, servidores, roteadores, protocolos e toda a arquitetura que permite a comunicação entre computadores pelo mundo, e existe desde a ARPANET.
A Web é um dos serviços que roda sobre a Internet, sendo um sistema de páginas e documentos interligados por hiperlinks acessados por meio de navegadores, utilizando protocolos como o HTTP.

A Internet é o sistema que permite a web funcionar.


### Arquitetura Cliente-Servidor

É o modelo fundamental organizador da comunicação na internet. Nessa arquitetura, há dois papéis principais:
O Cliente, que é geralmente o seu dispositivo, app ou navegador, faz a requisição.
O Servidor, que recebe a requisição, faz o processamento e devolve uma resposta.
Por exemplo: quando você pesquisa qualquer site no navegador google, o seu dispositivo, que é o cliente, manda uma requisição ao servidor do Google, que recebe, processa e devolve a página HTML com os resultados em segundos.

### IPs

IP, ou Internet Protocol, é o identificador único de cada dispositivo conectado à internet. É como o endereço digital de seu dispositivo.
Um exemplo seria quando você acessa um site no navegador. Quando você digita o nome do site, seu dispositivo Consulta um servidor DNS, traduzindo o nome legível do site para um endereço IP real, que é o identificador deste site em específico. Esse IP é usado para encontrar e acessar o servidor do Youtube.
