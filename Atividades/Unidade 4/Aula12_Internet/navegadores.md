# Navegadores

Um navegador é um software que atua como intermediário entre o usuário e a internet, sua principal função sendo interpretar e renderizar as três linguagens fundamentais da Web:


## Linguagens Web

### HTML
HyperText Markup Language, define a estrutura e o conteúdo da página, como título, imagens, links, botões;

### CSS
Cascading Style Sheets, Define a aparência, com cores, fontes, tamanhos, animações, espaçamentos, etc

### Javascript
Define o comportamento e a interatividade, com menus que abrem, formulários que podem ser validados, conteúdo que atualize sem precisar recarregar a página, etc.

## Motores de renderização
É o componente interno do navegador responsável por esse processo de interpretação e exibição, lendo o código HTML e CSS.
Cada navegador usa um motor diferente:

### Blink
Motor usado pelo Chrome e pelo Edge, é o mais utilizado no mundo atualmente, visto que o Chrome domina a maior parte do mercado de navegadores.
Foi criado em 2013 como um fork do WebKit, e é conhecido pela alta performance e atualização constante.

### Gecko

Motor desenvolvido pela Mozilla, e utilizado no Firefox. Um dos mais antigos que ainda está ativo, sendo criado em 1997. Caracterizado pelo forte compromisso com os padrões abertos da Web e a privacidade do usuário.
É independente das grandes big techs, e por isso é considerado importante para a diversidade do ecossistema web, evitando o monopólio de um motor apenas.

### WebKit

Motor desenvolvido pela Apple, e usado no Safari, que é o navegador padrão de todos os dispositivos Apple. Caracterizado pela sua eficiência energética, consumindo menos bateria que os outros motores, já que foi feito para navegar em dispositivos mobile, e a  Privacidade, mas é um ecossistema fechado, controlado pela Apple.
