# Startup para a Indústria de jogos brasileira.

## nome: BrGMarket (Brazilian Gaming Market)


Funcionará utilizando como inspiração a plataforma Steam, mas ela será específica para jogos brasileiros e acessível para jogadores brasileiros.

### Dados brutos coletados: 
  Preferências do usuário;
  Entrada: Cliques, tempo de sessão do usuário, buscas, reviews.
  Origem: O jogador
	Dado semi estruturado;
	
#### Jogos feitos; 
  Entrada: Jogo publicado; 
  Origem: Empresas ou Game Devs. 
  Dado estruturado;

#### Quantidade de cópias (digitais) vendidas.
  Entrada: Quantas transações foram realizadas para cada jogo;
	Origem: Transações completas;
	Dado estruturado.

#### Demografia dos jogadores
Entrada: Relação entre a localização dos usuários e os jogos jogados.
Origem: Localização.
Dado estruturado.

### Processamento
O sistema pegará todas essas informações e criará um algoritmo para cada usuário que usar a plataforma. 
Os dados serão coletados e armazenados em um banco de dados, processados por algoritmos de IA, e transformados em relatórios para serem analisados, juntamente com dashboards.

### Informação e conhecimento

São gerados padrões diversos, para cada usuário, customizando sua experiência.
Padrões como: Jogadores de X lugar, preferem fazer transações do tipo Y, e jogar jogos do tipo Z.
Tendências, como crescimento no número de usuários, aumento ou diminuição de jogos em um período de tempo, como: Jogos indies crescem em 100% Ao longo de 3 meses.
e baseado nesses padrões e tendências, as previsões poderão ser feitas.

E esses insights vão ajudar os usuários a customizar sua experiência, e vão ajudar os game devs também, quando fizerem seus jogos.

### Decisão e Valor
Tendo em vista todas as informações, podem ser consideradas soluções para os jogadores e game devs, fornecendo relatórios para devs, e uma experiência única para os jogadores
O valor principal, é a redução de custos tendo em vista o poder de compra do jogador médio brasileiro, e a personalização para cada usuário.
